Druid Tome - by Inner Drive Studios
-----------------------------------

Please refer to the Demo Scene to see the model in action.
By default the Prefab contains a correctly sized BoxCollider,
but no rotation script, this has only been added for 
demonstration purposes in the Demo Scene.

If you want to use this asset with URP, 
simply select the DruidTomeMaterial in the Materials folder and go to:

> Edit -> Rendering -> Materials -> Convert Selected Built-in Materials to URP <
